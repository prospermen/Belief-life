# 心理健康助手 - 情绪日志与正念练习应用

一款以情绪日志和引导练习为核心的跨平台心理健康移动应用，提供安全、温暖、宁静且富有鼓励性的用户体验。

## 🌟 核心功能

### 1. 新手引导页面
- **三页引导流程**：欢迎页、情绪记录介绍、正念练习介绍
- **精美插图**：温暖的视觉元素和动画效果
- **流畅过渡**：页面切换动画和分页指示器
- **呼吸动画**："开始使用"按钮的微妙动画效果

### 2. 用户认证系统
- **登录/注册界面**：简洁优雅的表单设计
- **表单验证**：实时验证和错误提示
- **社交登录**：微信、QQ登录选项
- **密码重置**：完整的密码找回流程

### 3. 情绪日志记录
- **情绪选择器**：6种核心情绪（开心、难过、愤怒、焦虑、平静、兴奋）
- **情绪强度**：1-10级滑块调节
- **文本编辑器**：支持长文本输入，实时字数统计
- **标签系统**：12个预设标签分类（工作、学习、家庭等）
- **表单验证**：确保数据完整性
- **保存功能**：带加载状态的保存操作

### 4. 引导练习系统
- **练习分类**：呼吸练习、冥想、放松、睡眠
- **练习列表**：4种不同类型的练习内容
- **音频控制**：播放、暂停、停止功能
- **呼吸动画**：实时的呼吸引导可视化
- **进度跟踪**：倒计时和进度条显示
- **练习提示**：实用的指导建议

### 5. 历史记录管理
- **时间线布局**：优雅的垂直时间线设计
- **情绪光谱**：彩色边框表示不同情绪
- **筛选功能**：按情绪类型筛选日志
- **详情查看**：完整的日志详情模态框
- **编辑删除**：日志管理功能
- **统计信息**：本周数据统计展示

## 🎨 设计规范

### 色彩方案
- **主色调**：柔和的蓝色 (#AEC6CF)
- **辅助色**：温暖的米色 (#F5F5F5)
- **点缀色**：活力的珊瑚色 (#FF6F61)
- **文字色**：深灰色 (#4A4A4A)

### 设计元素
- **圆角设计**：12-16px 大圆角
- **阴影效果**：轻微的卡片阴影
- **充足留白**：舒适的视觉间距
- **响应式布局**：适配不同屏幕尺寸

### 交互动画
- **微妙动画**：按钮点击反馈
- **平滑过渡**：页面切换效果
- **加载状态**：操作反馈动画
- **呼吸动画**：练习引导可视化

## 🛠️ 技术栈

- **前端框架**：React 18
- **构建工具**：Vite
- **样式方案**：CSS3 + Flexbox/Grid
- **状态管理**：React Hooks (useState, useEffect)
- **开发工具**：ES6+, 模块化开发

## 🚀 快速开始


```bash
npm run build
```
